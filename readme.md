To Run the app 

1) Install python or Anconda

2) Inside your code editor select the desired enviroment (recommended to create a separate virtual venv)

3) Then run this command in the terminal opened inside the project file
    "pip install -r requirements.txt"

4) Then after installing all dependencies in the same terminal run this command
    "python app.py"

5) to see the app in working, open your browser and go to http://127.0.0.1:5000 (or just click this link)
